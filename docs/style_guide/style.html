Naming Conventions

Files

all lowercase, words are separated by underscores.

Example

my_source_code_file.js
ansi_terminal.js

Justification:

use all lower case file names to avoid any problems on (non)case sensitive file systems.

Variables

Write variable names in camel case, shorter, more local names are encouraged.
Simple, less than 10 loc functions with one or two arguments can use single letter variable names if their function is obvious.
Variables that are used as constants (Written only once, and than only read) Should be written in all Capitals (Upper-case) with underscores.

Example

myVariableName
thisIsAGoodName
THIS_IS_A_CONSTANT
CONST_BY_WHY_AM_I_SHOUTING

Functions

Functions names should consist of lower case letters, words separated with underscores.

Example

my_function_name()
this_is_a_good_function_name_just_a_bit_long()

Justification:

Most modern style guides use camel case for function names, but that is a bad idea since you can't distinguish functions from variables, this is especially important since javascript has First-class functions. That can easily lead to "undefined is not a function".

Classes

Classes should use camel case names with should always start with upper case letter.

Example

MyClassName
ARandomClassName

Class Members

Members should be no different than any other variable

Whitespace

Indentation

Indentation should be done with tab characters, each indentation level is 1 tab character.
There should be no other whitespace character then tab before any non whitespace character.
There should be no tab character after any non whitespace character.
Always make sure your indentation is correct.

NEVER under any circumstance should a tab and a space character be beside each other!

Justification:

Using tabs to indent allows every developer to use their favorite indentation size without modifying the source code

Comments

Language

All comments should in english language

Code

Never leave unused, commented code in the source files, that's what version control is for.
